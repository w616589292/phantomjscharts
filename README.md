# phantomjscharts
基于phantomjs + highcharts/echarts 的服务器端丰富图表生成方案

# FAQ
若图表需要展示中文，需要安装中文字体库

    在centos中执行：yum install bitmap-fonts bitmap-fonts-cjk
    在ubuntu中执行：sudo apt-get install xfonts-wqy
